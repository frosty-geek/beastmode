# Release: iterm2-tmux

**Version:** v0.47.0
**Date:** 2026-03-30

## Highlights

Adds iTerm2 as a native dispatch strategy for pipeline visibility. The `it2` CLI drives tab-per-epic and split-pane-per-phase layout with automatic detection, lifecycle management, and startup reconciliation of orphaned sessions.

## Features

- iTerm2 dispatch strategy (`dispatch-strategy: iterm2`) using `it2` CLI for native tab/pane management
- `It2Client` typed wrapper for `it2` CLI commands (create-tab, split-pane, close-pane, list-sessions)
- `ITermSessionFactory` implementing `SessionFactory` interface with full pane lifecycle
- Auto-detection chain extended: iTerm2 -> cmux -> SDK when using `dispatch-strategy: auto`
- Environment detection via `ITERM_SESSION_ID` and `TERM_PROGRAM` env vars
- Startup reconciliation of existing `bm-*` tabs/panes from previous runs
- iTerm2 badge notifications for errors and blocked gates

## Fixes

- GitHub sync: runtime discovery replaces manual config IDs
