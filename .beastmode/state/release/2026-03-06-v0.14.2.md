# Release v0.14.2

**Date:** 2026-03-06

## Highlights

Retro system can now detect missing L2 context domains and propose new files seeded from session evidence, with confidence-scored promotion thresholds and a configurable HITL gate.

## Features

- **L2 gap detection** — Context walker gains Gap Detection Protocol with structured `context_gap` output type, confidence scoring (HIGH/MEDIUM/LOW), and accumulation-based promotion thresholds
- **Gap proposal processing** — Retro phase gains step 9 for processing context gap findings: logs all gaps to learnings, gates file creation via `retro.l2-write`, creates approved L2 files with session-seeded content
- **New HITL gate** — `retro.l2-write` gate (default: human) controls whether new L2 files are created automatically or require user approval

## Full Changelog

- Added `retro:` section to `config.yaml` with `l2-write: human` gate
- Extended `agents/retro-context.md` with Gap Detection Protocol (identify signals, score confidence, check accumulation, emit proposals)
- Updated `agents/retro-context.md` output types to include `context_gap` and `context_gap_logged`
- Updated `agents/retro-context.md` review rules to use gap detection protocol
- Added step 9 (Process Context Gap Proposals) to `skills/_shared/retro.md` with sub-steps for logging, gating, and file creation
- Renumbered Bottom-Up Summary Bubble from step 9 to step 10
