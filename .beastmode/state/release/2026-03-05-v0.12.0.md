# Release v0.12.0

**Date:** 2026-03-05

## Highlights

Beastmode now has a personality. A centralized persona definition gives all interactions a deadpan minimalist voice with context-aware session greetings.

## Features

- Centralized persona definition at `skills/_shared/persona.md` — character bible with voice rules, tone guardrails, and context-awareness
- Expanded session-start tagline pool from 14 to 34 deadpan lines
- All 5 workflow skill announces replaced with persona-voiced equivalents via @import
- CLAUDE.md Prime Directive updated to adopt persona for all interactions
- Context-aware greetings factor in time of day and project state
