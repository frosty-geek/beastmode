# Design Meta

## Process
- ALWAYS produce dated research artifacts from 3+ external sources before locking structural decisions — research-informed designs outperform brainstormed ones
- ALWAYS enumerate every instance in concrete tables for N-instance decisions — catches edge cases that pattern descriptions miss
- ALWAYS start from existing algorithms when building structurally analogous subsystems — constrains design space and ensures consistency
- ALWAYS include shared files (_shared/) in phase-scoped sweeps — shared files are blind spots for phase-scoped refactors
